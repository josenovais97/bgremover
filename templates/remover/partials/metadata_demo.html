{% comment %} Metadata-stripping demo, for the EXIF remover. Stripping EXIF changes nothing you can see — the pixels are identical, which is the whole point (it's lossless). So there is no before/after image to compare; the thing that changes is the data, and that's what this shows: the real fields a photo carries, struck through as they're removed. No parameters — the field list is the demo. Purely markup, so there's no asset to keep in sync with it. The values are the REAL metadata inside static/img/demo-exif-sample.jpg, which is the same file the page's "Try a sample photo" button loads. They used to be an invented iPhone-in-Lisbon set, so the demo and the sample contradicted each other the moment anyone clicked it. {% endcomment %}
