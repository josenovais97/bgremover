{% comment %} Metadata-stripping demo, for the EXIF remover. Stripping EXIF changes nothing you can see — the pixels are identical, which is the whole point (it's lossless). So there is no before/after image to compare; the thing that changes is the data, and that's what this shows: the real fields a photo carries, struck through as they're removed. No parameters — the field list is the demo. Purely markup, so there's no asset to keep in sync with it. {% endcomment %}
