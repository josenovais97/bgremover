{% load static %} {% comment %} Reusable before/after comparison demo (drag-to-reveal slider). Include it and pass these keyword params: before, after - static image paths, e.g. "img/demo-blur-before.webp" before_label - badge shown top-left (the original) after_label - badge shown top-right (the result) caption - small helper line under the frame checker - truthy to show the transparency checkerboard (for cut-outs) aspect - Tailwind aspect class, defaults to landscape "aspect-[4/3]". Pass the literal class (e.g. aspect='aspect-[3/4]') so Tailwind's scanner still sees it in the including template. initial - starting split, 0-100 (default 50). Lower shows more of the "after" at rest — useful when the effect sits off-centre and a 50/50 split would hide it until the user drags. Driven by static/js/demo.js, which wires every [data-demo] block on the page. {% endcomment %}
